# Portfolio Website

Welcome to my personal portfolio website! This repository contains the source code for my portfolio, showcasing my featured projects, technical skills, and experience.

## 🚀 Tech Stack

This website is built with modern web technologies focusing on performance, aesthetics, and smooth interactions:

- **Framework**: [Next.js (v16)](https://nextjs.org/)
- **UI Library**: [React (v19)](https://react.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS (v4)](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Deployment**: [Vercel](https://vercel.com/)

## 📂 Featured Projects Included

- **Nigerian Job Market Analyzer**: A complete data analysis pipeline built to collect, clean, and analyze Nigerian tech job listings.
- **Automated Data Quality Firewall**: An automated system designed to validate, clean, and monitor incoming data pipelines.
- **Marprom Stores**: A full-stack e-commerce web application for premium organic apparel and accessories with WhatsApp checkout.
- **Codveda Internship Tasks**: Data Analytics projects focusing on Exploratory Data Analysis (EDA) and data cleaning.

## 💻 Getting Started Locally

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
